Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

1) Изменить режим запуска по умолчанию ОС. + 
2) Переключиться между несколькими режимами работы. + init 1, 2, 3, 4,..
3) Убрать из автозапуска по умолчанию службу сети. +   service network stop
4) Проверить статус службы аудио.+  sudo systemctl status audio
5) Перезагрузить ПК через работу с уровнями выполнения или таргетами.+ reboot
